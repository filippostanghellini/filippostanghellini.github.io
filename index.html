Experience

2023

University Research Assistant — Data Scientist (Internship)

    Developed a Cognitive Network based on the items of the DASS questionnaire (Depression Anxiety Stress Scale) using Python and the EmoAtlas library to build a semantic-syntactic network. Performed graph analysis in R (degree, closeness, modularity, assortativity) and applied various clustering techniques to identify main conceptual communities. In the second phase, built a Psychometric Network using the EGAnet package to estimate the number of factors and assess their stability through bootstrap methods, comparing results with the theoretical structure of the DASS. Finally, conducted a comparative analysis between the two networks using the Jaccard Index, highlighting a significant overlap between semantic communities and psychometric factors.
